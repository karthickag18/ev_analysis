# Electric Vehicle Population Data Analysis – Tableau

A Tableau dashboard analyzing the Electric Vehicle (EV) population dataset to track EV adoption trends by state, make, model, and vehicle type.

## Dashboard

**ELECTRIC VEHICLE ANALYSIS** – single-page dashboard covering:

- Total Vehicles, BEV Vehicles, PHEV Vehicles (with % share)
- Vehicles by Make and Model (Top N)
- Average Electric Range
- Vehicles by Model Year
- Vehicles by State
- CAFV (Clean Alternative Fuel Vehicle) Eligibility

## Dataset

`Electric_Vehicle_Population_Data.hyper` — registered EV records including:

- Make, Model, Model Year
- Electric Vehicle Type (BEV / PHEV)
- Electric Range
- CAFV Eligibility
- State, County, City, Postal Code

## Tools Used

Tableau Desktop, Tableau Hyper Extract

## How to View

1. Install [Tableau Desktop](https://www.tableau.com/products/desktop) or [Tableau Reader](https://www.tableau.com/products/reader) (free)
2. Open the `.twbx` file
3. Explore the Electric Vehicle Analysis dashboard
